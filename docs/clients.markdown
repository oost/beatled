# Clients

Beatled ships three client apps, all using the same [HTTPS API](../api.html) to monitor and control the system.

| Client | Technology | Description |
|--------|-----------|-------------|
| [React (Web)](client.html) | React 18, TypeScript, Vite | PWA dashboard for any browser or phone |
| [iOS & macOS](ios.html) | SwiftUI, Swift Charts | Native universal app for iPhone and Mac |
